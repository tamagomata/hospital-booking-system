# 病院予約システム

このリポジトリは、個人開発による病院予約システムのソースコードです。  
卒業研究の一環として、以下の技術スタックを使用して構築しました。

## 使用技術

- **フロントエンド:** HTML, CSS, JavaScript
- **バックエンド:** Node.js
- **データベース:** MongoDB
- **開発ツール:** Visual Studio Code, Git, GitHub

## ディレクトリ構成

```plaintext
hospital-booking-system/
├── server.js
├── package.json
├── /config
│   └── db.js
├── /models
│   └── bookingModel.js
├── /controllers
│   └── bookingController.js
├── /routes
│   └── bookingRoutes.js
└── /frontend
    ├── index.html
    ├── /css
    │   └── style.css
    └── /js
        └── main.js
````
